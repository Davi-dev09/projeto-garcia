Coloque aqui as imagens reais do Instagram da Garcia Engenharia & Construções.

Sugestão de nomes de arquivos para combinar com o site:

- logo-garcia.png                -> logo em fundo transparente
- hero-obra-principal.jpg        -> foto em destaque de uma obra (banner principal)
- sobre-equipe-obra.jpg          -> foto da equipe em obra
- sobre-planejamento.jpg         -> foto de planejamento / plantas / escritório
- portfolio-residencial-1.jpg    -> obra residencial 1
- portfolio-residencial-2.jpg    -> obra residencial 2
- portfolio-comercial-1.jpg      -> obra comercial 1
- portfolio-comercial-2.jpg      -> obra comercial 2
- portfolio-reforma-1.jpg        -> foto de reforma / antes e depois

Passos:
1) Baixe as imagens do Instagram (ou do seu acervo) para o computador.
2) Renomeie cada imagem conforme os nomes acima.
3) Coloque todos esses arquivos dentro desta pasta "assets".

Quando esses arquivos existirem aqui, o site já vai usar automaticamente as imagens nas seções de:
- Banner principal (Home)
- Sobre Nós
- Portfólio
